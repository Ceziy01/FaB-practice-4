# 🎮 Магазин видеоигр

Приложение для управления списком товаров интернет-магазина видеоигр.  
Реализован базовый CRUD-функционал с подключением к API.

---

# 📄 Описание страницы

Главная страница приложения представляет собой интерфейс управления товарами.

Пользователь может:

- ➕ Добавлять новый товар
- 📋 Просматривать список товаров
- ❌ Удалять товар

Все изменения отображаются на странице без перезагрузки.
