# 🎮 Магазин видеоигр

Проект состоит из:

- Backend: Express + Swagger
- Frontend: React
- Стилизация: SCSS
- Работа с API: Axios

# Описание проекта

Приложение представляет собой простой CRUD-магазин видеоигр.

Пользователь может:

- Добавлять товар
- Просматривать список товаров
- Получать товар по ID
- Обновлять товар
- Удалять товар

Все операции выполняются через REST API.

# Frontend (React)

Клиентская часть реализована на **React**.

## Используется:

- `useState` — управление состоянием
- `useEffect` — загрузка данных при монтировании
- `axios` — HTTP-запросы
- SCSS — стилизация

## Функционал:

- Отображение списка товаров
- Форма добавления товара
- Удаление товара без перезагрузки страницы
- Динамическое обновление интерфейса

---

# Backend (Express)

Серверная часть реализована на **Express.js**.

## Используется:

- express — создание REST API
- cors — разрешение кросс-доменных запросов
- nanoid — генерация уникальных ID
- swagger-jsdoc — генерация OpenAPI документации
- swagger-ui-express — интерфейс документации

## Реализованные эндпоинты:

| Метод | Endpoint | Описание |
|--------|----------|----------|
| GET | /api/products | Получить список товаров |
| GET | /api/products/:id | Получить товар по ID |
| POST | /api/products | Создать товар |
| PATCH | /api/products/:id | Обновить товар |
| DELETE | /api/products/:id | Удалить товар |
